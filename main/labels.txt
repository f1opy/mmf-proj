Список широкоуживаних ярликів для рівнянь. 

Замість ## треба вставити номер задачі, до якої відноситься рівняння. Наприклад, якщо у вас є математична умова задачі №1.1, на яку треба послатися в тексті, то використовуєте ярлик cond1,1 

1. cond## -- умова задачі
2. subst## -- підстановка, яка дозволяє розв'язати задачу (зазвичай це розділення змінних u(x,t) = X(x)T(t) )
3. sepvar## -- результат розділення змінних
4. ShLsol## -- розв'язок задачі Штурма-Ліувілля
5. mode## -- формула для u_n(x,t)
6. gensol## -- формала для u(x,t) для довільних коефіцієнтів 
7. (sol##)_t -- похідна по часу від \ref{gensol##}
8. init-pos## -- результат підстановки \ref{gensol##} в умову u(x,0)
9. init-vel## -- результат підстановки \ref{gensol##} в умову u_t(x,0)
10. orth## -- співвідношення ортогональності  
11. fourier-coefs## -- формула обчислення коефіцієнтів розкладу в ряд Фур'є
12. Cauchy-sol## -- єдиний розв'язок u(x,t) визначенний з початкових умов
13. new-cond## -- умова задачі після заміни змінної
